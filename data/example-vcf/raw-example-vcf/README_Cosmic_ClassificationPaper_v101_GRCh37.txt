---------------------------------------
COSMIC Paper Classification Information
---------------------------------------
  COSMIC paper cancer classification information in a tab separated file.  [ Cosmic_ClassificationPaper_v101_GRCh37.tsv.gz ]

  File Description

[column number:label] Heading                          Description
--------------------------------------------------------------------------------------------------------
[1:A]                COSMIC_PHENOTYPE_PAPER_ID         A unique COSMIC identifier (COSOP) for the paper classification.
[2:B]                COSMIC_PHENOTYPE_ID               A unique COSMIC identifier (COSO) for the classification. This identifier can be used to retrieve tissue and histology information from the classification file.
[3:C]                PAPER_PRIMARY_SITE                Primary tissue specified in the original paper.
[4:D]                PAPER_SITE_SUBTYPE_1              Sub tissue specified in the original paper.
[5:E]                PAPER_SITE_SUBTYPE_2              Sub tissue specified in the original paper.
[6:F]                PAPER_SITE_SUBTYPE_3              Sub tissue specified in the original paper.
[7:G]                PAPER_PRIMARY_HISTOLOGY           Primary histology specified in the original paper.
[8:H]                PAPER_HISTOLOGY_SUBTYPE_1         Sub histology specified in the original paper.
[9:I]                PAPER_HISTOLOGY_SUBTYPE_2         Sub histology specified in the original paper.
[10:J]               PAPER_HISTOLOGY_SUBTYPE_3         Sub histology specified in the original paper.
